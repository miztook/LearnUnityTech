功能说明
	GUnlimitedList 无限循环列表
	它可以对指定模板(item)进行复制,个数有程序指定。
	并且按照横向或者纵向的顺序排列, 可以指定宽高（行数列数）。
	程序可以对某个索引下的item进行操作，根据索引进行差异化处理。
	它可以回收移出视口的item并且进行重复利用。所以它更适用于数量较多的列表，例如背包，商店。


结构和使用说明
	一个完整的GUnlimitedList有三个节点，从父级到子级分别是：1）视口层。2）布局和容器层。3）item模板层。（图）

	1）视口层。
		就是显示区域的一个矩形，无论它的子级包含什么内容，只要超出这个范围都不会被显示。

			Image		根据情况可以挂或者不挂，这个Image的作用就是接收鼠标事件的。所以它的raycast target属性一定要勾选上。
					例如：有的列表排列很松散，当拖动Item之间的空白区域会有接收不到鼠标事件的情况，此时需要挂Image。

			ScrollRect	必挂，此组件可以让你选定一个目标作为可滑动的对象，并且提供了一些参数。
					具体参数的意思，可以查看Unity官方文档。
					
			RectMask2D
					必挂，此组件是做裁切用的。具体参数的意思，可以查看Unity官方文档。

	2）布局和容器层。

			GUnlimitedList	必挂，列表的核心逻辑都在这里。

			参数解释：
				Number Of Row		行数
				Number Of Column	列数
				Cell			模板
				Spacing			间隔
				Direction		滑动方向

	
	3）item模板层。一个会被复制多个的模板。（GListItem不用挂）
